Project_Jindong_website
本專案練習主要為模擬京東主頁與詳情頁部分功能。

涉及技術工具如下:
1. HTML
2. SCSS
3. Javascript
4. NodeJS (建立簡易的後端服務器)
5. NPM

亮點包含:
1. 模糊查詢
2. Axios發送請求
3. Throttle 節流 (lodash插件)
4. Lazy loading 圖片懶加載 (IntersectionObserver API)
5. 輪播圖計時器詳情頁放大鏡 等小動畫
6. JS代碼的模塊化管理

備註:
1. 下載後，請先啟動server，否則網頁部分功能將無法獲取數據

圖片來源: 京東官網
